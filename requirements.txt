# --- pandas-ta notes ---------------------------------------------------------
# REQUIRES Python 3.9 (3.9.7 is the tested version; 3.10 also works).
# pandas-ta 0.3.14b0 does NOT work on Python 3.11+ — `pip install pandas-ta`
# pulls a build that fails to import there. Create the venv with Python 3.9:
#     py -3.9 -m venv venv
# If `pip install pandas-ta` still fails (the 0.3.14b0 build was pulled from
# PyPI for some Python versions), install from source instead:
#     pip install git+https://github.com/twopirllc/pandas-ta.git@0.3.14b0
#
# Optional: TA-Lib (the `talib` package) is more robust than pandas-ta but needs
# the native TA-Lib C library installed first. It is NOT required to run the
# framework. Install only if a strategy imports `talib`:
#     pip install TA-Lib
